# API Ссылка

## Обзор

API предоставляет REST эндпоинт для функциональности ИИ-агента.

## Эндпоинт

### POST /run

Обрабатывает пользовательский ввод через ИИ-агент и возвращает ответ.

**URL**: `http://127.0.0.1:8000/run`

**Метод**: POST

**Content-Type**: application/json

**Тело Запроса**:
```json
{
  "user_id": "string",
  "text": "string"
}
```

**Ответ**:
```json
{
  "answer": "string"
}
```

**Коды Статусов**:
- 200: Успех
- 422: Ошибка валидации
- 500: Внутренняя ошибка сервера

**Пример**:
```bash
curl -X POST "http://127.0.0.1:8000/run" \
     -H "Content-Type: application/json" \
     -d '{"user_id": "123456", "text": "Привет"}'
```

Ответ:
```json
{
  "answer": "Привет! Чем могу помочь сегодня?"
}
```

## Технические Детали Реализации

- Построен с FastAPI
- Асинхронная обработка
- Вызывает ядро агента с пользовательским вводом
- Возвращает LLM-сгенерированный ответ
- Без аутентификации (для разработки)
